태그 div

border

border(테두리 스타일)
solid : 실선의 형태로 적용함
dotted : 점선의 형태로 적용함
double : 이중 선의 형태. 굵기가 3px 이상이 되어야 볼 수 있음
groove : 입체적으로 선을 홈이 들어간 것처럼 보여줌, 최소 2px은 되어야 확인 가능
ridge : groove와 비슷하지만, 음영 값이 반대라 선이 돌출되어 보임
inset : 요소 전체가 안으로 들어가 보임
outset : inset의 반대,요소 전체가 밖으로 돌출되어 보임
inherit : 부모의 속성을 상속받음



id,class

id는 스타일을 지정할 때 한가지만 지정해서 쓰는 이름
쓰는법
#아이디{attribute,value;attribute2;value;}
div id = "아이디"

class는 그룹으로 묶어서 스타일을 지정할때 쓰는 이름
쓰는법
.클래스명{attribute:value;attribute2;value;}
p.클래스명 {attribute:value;attribute2;value;} p태그에 있는 클래스에서 사용 가능
* { attribute;value,attribute2;value} 모든 태그에서 사용함
div class = "클래스명"
